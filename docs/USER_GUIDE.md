## Getting Started

### What is PuzzleBITE?
PuzzleBITE is an innovative gamified dining experience platform that combines puzzle-solving with restaurant rewards. Customers solve category-based puzzles to unlock discounts and special offers at partner cafes.

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Mobile or desktop device

### First Time Setup
1. Open the PuzzleBITE application in your browser
2. The app will automatically detect your location and show nearby partner cafes
3. No registration required - start playing immediately!

## Troubleshooting

### Common Issues

#### Puzzle Timer Issues
**Problem**: Timer continues after completing puzzle
**Solution**: 
- Navigate back to home screen
- Timer automatically stops on puzzle completion
- Refresh app if timer persists

#### Offer Not Activating
**Problem**: Cannot activate offer
**Solution**:
- Check internet connection
- Ensure you're at the correct cafe location
- Contact cafe staff for manual activation

#### Charts Not Loading
**Problem**: Statistics charts don't display
**Solution**:
- Refresh the page
- Check browser compatibility
- Clear browser cache

#### Theme Not Switching
**Problem**: Theme toggle doesn't work
**Solution**:
- Tap the toggle button in header
- Refresh app if needed
- Check if JavaScript is enabled

### Performance Tips
